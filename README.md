# User Profile Service

A Spring Boot microservice for managing user profile records in a larger system.

## API

Base URL:

```text
http://localhost:8080
```

API name:

```text
User Profile Service
```

## Operations

| Method | Path | Description |
| --- | --- | --- |
| `POST` | `/api/v1/profiles` | Create a profile |
| `GET` | `/api/v1/profiles/{id}` | Get a profile by ID |
| `GET` | `/api/v1/profiles/by-username/{username}` | Get a profile by username |
| `GET` | `/api/v1/profiles?status=ACTIVE&page=0&size=20` | List profiles with optional status filter |
| `PATCH` | `/api/v1/profiles/{id}` | Update profile fields |
| `PATCH` | `/api/v1/profiles/{id}/deactivate` | Mark a profile inactive |
| `DELETE` | `/api/v1/profiles/{id}` | Soft-delete a profile |

## Run Locally

Requirements:

- Java 17+
- Gradle 8+

```bash
gradle bootRun
```

The API runs on `http://localhost:8080`.

Swagger UI is available at `http://localhost:8080/swagger-ui.html`.

H2 console is available at `http://localhost:8080/h2-console`.

Health check is available at `http://localhost:8080/actuator/health`.

## Profile Entity

| Field | Type | Description |
| --- | --- | --- |
| `id` | UUID | Unique profile identifier generated by the service |
| `username` | String | Unique human-readable user handle |
| `email` | String | Unique contact email |
| `firstName` | String | User first name |
| `lastName` | String | User last name |
| `phoneNumber` | String | Contact phone number |
| `displayName` | String | Public display name |
| `jobTitle` | String | User job title |
| `department` | String | User department or team |
| `location` | String | User location |
| `dateOfBirth` | Date | User date of birth |
| `bio` | String | Short profile biography |
| `avatarUrl` | String | Profile image URL |
| `preferredLanguage` | String | Preferred language code |
| `timeZone` | String | Preferred time zone |
| `status` | Enum | `ACTIVE`, `INACTIVE`, `SUSPENDED`, or `DELETED` |
| `createdAt` | Timestamp | Profile creation time |
| `updatedAt` | Timestamp | Last profile update time |
